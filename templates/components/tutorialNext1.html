Errors that can happen when generating


Error in generating file!
this means that your file is not an .sb3 file.



Invalid URL or internal error!
 URL is incorrect or scratch is having server issues while you are generating your project.

When your project is generated this box will show up explaining what PyStage has not implemented yet.
