Beginner Tutorial 
            


How to make a basic scratch project.


1. File -> New.

2. Click choose a sprite in the bottom right corner.

3. search for dot in the search bar and click on dot the dog.

4. Next delete the cat by clicking on the trash can on the sprite.

5. Click on Events.

6. Drag When green flag clicked over to the script area.

7. Click on Motion.

8. Drag move 10 steps over to the script area.

9. Click on Sounds.

10. Drag start sound bark over to the script area.

11. Make sure all 3 steps are connected to each other in this exact order

12. Press the green flag to run your project.
